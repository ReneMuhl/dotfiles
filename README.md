# My dotfiles

## including

* bash
* git
* vim
* zsh
* sublime text 3

## installation

1. Clone the repository

`git clone --recursive https://github.com/rm--/dotfiles`

2. Backup your dots and install my dots with

`make install-dots`

3. (If you don't like it and want you old dots back), run

`make restore`
